    Output

    This demo uses multiple API calls for its functionality. The tone API analyzes and displays the Emotion, Social, and Writing tone results.

    Then when you click a highlighted word, a call to the synonyms API displays suggested synonyms that either soften or strengthen the tone of that word. It also allows you to select a synonym to replace the original word, changing the tone of the message. For an in-depth explanation of this service, visit the Documentation.

    • Percentile
    • Word Count
    • JSON

    Percentile compares the raw output score to a preset scorecard. For this demo the scorecard is Business Email.

    Word Count returns the percentage of words per tone and the total number of words for each individual tone trait.
